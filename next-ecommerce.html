NEXT – Ecommerce

Why This Stack?

Like I mentioned before, the goal of learning Next.js wasn’t just to build fancy static sites — it was to deliver real web apps too. With Next.js, I can build prototypes or even full apps much faster than with a regular MERN stack.

What I Learned

  • Improved my AuthJS knowledge
  • Got better at handling payments with Stripe and PayPal
  • Learned how to use React Email templates with Resend for beautiful emails
  • Solidified my flow using ShadCN, Zod, and React Hook Form
  • Sharpened my skills working with Prisma ORM

Cool Touches

  • User/admin auth roles
  • Admin panel/dashboard
  • Clean dark/light mode toggle

Future Ideas

  • Add more user roles so other admins can manage products, plus a super admin role
  • Add buttons/features for smoother navigation
  • Implement OAuth with NextAuth social providers
  • Continue improving the auth system overall
