AI Customer Support Platform couse (Code with Antonio)

Overview

In this course, we build an AI-powered customer support platform from the ground up. You’ll set up real-time AI chat with Convex Agents, add voice support with VAPI, and upload docs to create a knowledge base using embeddings and RAG. We teach the AI when to escalate a conversation to a human or auto-resolve it. You’ll also build workspaces, teams, authentication, and billing with Clerk, embed a chat widget, securely store API keys with AWS, and track errors with Sentry. Each chapter wraps up with a pull request and a CodeRabbit review.

What I’ll Learn

  • Real-time AI chat using Convex Agents
  • Voice support integration with VAPI
  • Document uploads → embeddings + RAG knowledge base
  • AI policies for auto-resolve vs human escalation
  • Workspaces, teams, authentication, and billing with Clerk
  • Embedding and customizing a chat widget
  • Secure API key storage with AWS
  • Error monitoring and tracing with Sentry
  • PR-driven workflow: each chapter ends with a PR + CodeRabbit review

Highlights

  • Smart escalation rules: seamless handoff to humans when needed
  • Fast RAG responses from an up-to-date, doc-backed knowledge base
  • Drop-in chat widget that’s easy to embed and configure
  • Clean team/workspace model for multi-user collaboration
  • Production-ready setup: Clerk auth + AWS secrets + Sentry visibility

Future Ideas

  • Expand escalation logic with SLAs and queue routing
  • Add analytics for resolution time, deflection rate, and CSAT
  • Multi-tenant billing refinements and usage-based pricing
  • Experiment with tool use / MCP after the core build
