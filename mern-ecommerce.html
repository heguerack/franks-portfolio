MERN – E-commerce

Why This Stack?

I wanted to combine my growing knowledge of Node/Express with front-end development. I wanted the ability to run my own server and implement backend logic. As I continue improving my JavaScript skills, this project felt like a natural next step.

What I Learned

  • Designing and structuring REST API routes
  • Connecting front end and back end for two-way communication
  • Server-side validation, JWT authentication, and authorization
  • Working with schemas and database operations
  • Deploying to Heroku

Cool Touches

  • User roles with admin authorization
  • Smooth dark/light mode toggle

Future Ideas

  • Add rate limiting and throttling to prevent abuse and reduce data transfer.
  • Add OAuth authentication and increase admin roles/levels of authorization; ideally no libraries involved other than jsonwebtoken (JWT) and OAuth.
  • Add React Query for powerful data fetching and caching
  • Eventually integrate an AI assistant to help track job applications and give suggestions
