Nodemailer

My go-to for sending emails from the backend — no fancy dashboards, just full control.

Nodemailer is great when I want to send transactional emails directly from a server. It’s simple, flexible, and works well with Gmail, Mailtrap, or any SMTP service. I like using Resend for production projects because it handles deliverability and has a nice UI — but when budget matters, Nodemailer gets the job done for free.

  • Works great with Express or any Node backend
  • No account needed — full email control via SMTP
  • Resend is nice, but Nodemailer is budget‑friendly

If I’m bootstrapping or testing, Nodemailer is the easiest way to send emails fast.
