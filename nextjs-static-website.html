Next.js - Static Website

Why This Stack?

After practicing with the MERN stack, Next.js felt like the right next step. It let me use modern features while delivering both static and dynamic pages.

What I Learned

  • Merged server and client components to add interactivity while still delivering static content — using wrappers and children where needed.
  • Improved my thinking around how content renders and loads, and how to optimize it for speed and user experience without hurting SEO.
  • Learned how to use static regeneration to load data on static pages directly from the server and rebuild the page periodically — getting the best of both static and dynamic rendering.

Cool Touches

  • Calendly integration for easy bookings
  • A project gallery, with each card having its own mini-gallery
  • Real SEO improvements — the site is live and actually bringing in business!
