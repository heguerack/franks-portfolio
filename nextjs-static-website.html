Next.js - Static Website

Why This Stack?

After practicing with the MERN stack, Next.js felt like the right next step. It lets me use modern features while delivering both static and dynamic pages.

What I Learned

  • Improved my merging of server components with client components to add interactivity while still delivering static content — using wrappers and children where needed.
  • Improved my critical thinking on how pages and content render, and how to optimize them for speed and user experience without sacrificing SEO.
  • Started learning how to work with other teams using Git.

Cool Touches

  • Calendly integration for easy booking
  • A gallery with project cards, each containing its own mini-gallery
